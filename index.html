Javascript Colors and Image Demo

This is a demo website made to demonstrate Javascript development abilities. The image below is made of 32,768 cells. Each cell contains one color. The colors have been programmatically generated by taking combinations of red, green and blue colors. Each color ranges over 32 values from 7 to 255 in steps of 8.
