Javascript Colors and Image Demo

This is a demo website made to demonstrate Vinnie's ability to code in Javascript. The image below is made of 32,768 cells. Each cell contains one color and has a height of 2 pixels and a width of 2 pixels. The colors have been programmatically generated by taking combinations of red, green and blue colors. Each color ranges over 32 values from 7 to 255 in steps of 8.
There are two options for presentation of generated colors. With sorting, the colors have been sorted on a ranking value generated by the values of the colors themselves.
